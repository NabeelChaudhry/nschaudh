Reflections

Before I started actually coding this project I had assumed I would struggle more with the animation aspect of Tetris rather than the game theory. However, once I started coding I realized that I found the game theory to be more difficult than the animations, despite making a few games prior to this. Learning the animation aspects through understanding how the event loop works and using the Java swing library was an awesome experience. Coding this project was challenging at times, but once it all came together and the game was running smoothly, it felt great!

Overall I learned a lot about animation in Java and also learned to appreciate how hard it is to make a game.

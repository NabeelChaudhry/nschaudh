Reflections

This being my first web application I made a lot of mistakes and in my code I focused more on functionality rather than organization (although my code is still is still organized, it could be better!). Learning to work with JSON and AJAX was an enlightening experience. Initially I had difficulty dealing with the asynchronous nature of getting and posting data to the web. When I first started coding I had AJAX code that changed local variables, and once I accessed these variables I wondered why the Chrome console would always display them as null. Once I realized that I could use the .done() function that executes code once the AJAX request was completed, it made my life a lot easier. It was a dumb mistake from me by not realizing the asynchronous nature of the post/get calls, but it was all a part of the learning experience!

Apart from what I said earlier, I learned quite a bit about using CSS and JQuery to enhance the look and feel of my app ( I tried to keep Spotify's black and green theme throughout my app). Although it looks a little rough, I'm pretty proud of it since I wrote essentially all of the CSS/JQuery myself. Furthermore, this was my first time working with an API and it was really cool learning how to get,post and interpret the data from Spotify's API!

Reflections

After learning React.js I thought it would be fun to make a weather web application. I initially was using the open weather map API, but ran into mixed content errors when hosting my app via Firebase. This was because open weather map API uses an HTTP protocol instead of HTTPS. Due to this I switched API's and used the apixu weather API instead. Before I even started I sketched out what I wanted my app to look like (all the images are made in photoshop):

I then recreated the weather card I made above in codepen. After this it was just using my React.js knowledge to make the individual components (search component, weather-card component, etc.) and adding some CSS & jQuery to make it look pretty. Unfortunately because of the API I was restricted to only letting the user search via postal codes. I could have also allowed searches based on longitude and latitude but I felt that postal codes are significantly more user friendly. I also had some fun using animations to make the spinning logo and the hover effect on the each weather card.

Overall I had a lot of fun using React.js and I'm excited what else I can do with React!
